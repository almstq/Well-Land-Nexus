# Well Land Ops v3.0 - Frontend Web Application

This folder is reserved for the upcoming fully responsive React + Vite frontend web application. 

## Features Planned for the Frontend
1.  **Multi-Role Viewports**:
    *   **Admin Dashboard**: Master control over fleet locations, lease billing summaries, and user registries.
    *   **Procurement Hub**: RFQ management, quote uploads with Gemini-driven automatic analysis, PO downloads, and ledger reconciliation.
    *   **Supervisor Panel**: Fleet meter hour logging, operator assignments, and live status reports.
    *   **Requestee Desk**: Generic item request submission and order tracking.
2.  **State Management**: Real-time integration with Google Sheets via the Apps Script API Gateway.
3.  **Modern UI/UX**: Dark mode, dynamic cards, glassmorphic layout, and rich animations.
